# crewzap-auto-post
Instagram Auto-Post Creator (CrewAI + Zapier + Buffer Scheduler)

This project is an AI-powered automated Instagram post generator built using CrewAI. It produces Instagram-ready captions, hashtags, and processed media, then sends the results to a Zapier Webhook, where the content is scheduled and published through Buffer—with zero need for Meta API setup.

 Core Features

 Multi-Agent AI Workflow that collaborates to create professional Instagram posts

 AI Image Captioning & Enhancement using Vision + Text Models

 SEO & Trend-Optimized Hashtags customized to niche and audience

 Simple Integration via Zapier Webhooks (no Meta Dev account required)

 Automatic Scheduling & Publishing through Buffer

 Brand Tone Memory for consistent style across posts

 Fully Extendable Agents & Tools for analytics, multi-account posting, templates, etc.

 Tech Stack
Component	Technology

 Agents Included

Caption Creator Agent – writes engaging, emoji-optimized captions

Hashtag Strategist Agent – generates niche-specific trending hashtags

Publisher Connector Agent – formats & sends content to Zapier for Buffer scheduling
